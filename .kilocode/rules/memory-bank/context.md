## Database

- **SQLite** via `better-sqlite3` (Bun-compatible)
- **Migrations**: `src/db/migrations/` (auto-generated by drizzle-kit)
- **Schema**: Users, Memories, Echoes, Tags, Tag Assignments (many-to-many)
- **Indexes**: Email, user_id, memory_id, name for performance

## API Status

All endpoints tested and working:
- `POST /api/auth/register` – User registration
- `POST /api/auth/login` – JWT issuance
- `GET /api/memories` – List user memories
- `POST /api/memories` – Create memory (uploads to Pastebin, triggers async echo)
- `POST /api/echoes` – Generate AI insight for a memory
- `POST /api/synthesis` – Synthesize multiple memories

Authentication via `Authorization: Bearer <token>` header.

## Known Issues

- No rate limiting on auth endpoints (brute-force risk)
- Pastebin single point of failure (consider backup persistence)
- Model download ~300MB on first run (cached in `.model-cache/`)
- AI quality lower than paid GPT/Claude but sufficient for reflection
